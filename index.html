Tests können deaktiviert werden: 

      class DisabledTests {

        @Test
        @Disabled
        void breakingTest() {
          fail("This never works!");
        }

        @Test
        @DisabledOnWeekends
        void breaksOnWeekends() {
          fail("Meeh, it's weekend again!");
        }
      }
